v0.3

This release introduces following changes in Murl class interface.

  • Add proxy support:
    • New public method set_proxy() to enable/disable proxy.
  • Improve server response format handling:
    • Changes in Murl class interface:
      • Type of format parameter in the constructor changed to integer: MURL_RESPONSE_FORMAT_XML = 0 or MURL_RESPONSE_FORMAT_JSON = 1.
      • Type of attribute 'status' changed to integer: MURL_STATUS_OK = 0, MURL_STATUS_ERROR = 1 and MURL_STATUS_UNKNOWN = 255.
    • New classes:
      • MurlResponseFormat - base server reply format.
      • MurlXmlResponse - XML response.
      • MurlJsonResponse - JSON response. Parsing not yet implemented.
  • Drop API method handling.
  • Update example code pymurl.py according to the changes in PyMurl.
