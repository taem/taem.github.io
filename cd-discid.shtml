cd-discid

cd-discid is an utility to get CDDB discid information from a CD-ROM disc.

Download

Latest version - 1.4: cd-discid-1.4.tar.gz.
Old version available here - 0.9: http://lly.org/~rcw/cd-discid/.

Development

Project page at github: https://github.com/taem/cd-discid.
Clone Git repository: git clone https://github.com/taem/cd-discid.git.

Maintainer
